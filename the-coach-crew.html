The Coach Crew

Web App



Brief

The Coach Crew is a start up focused on providing access to sports coaching and guidance across the country. The initiative was looking to build a simple, highly functional web application that directly connects users to highly skilled trainers and sporting centers. The application would feature dynamic elements such as scheduling direct meetings with coaches, listings map, and online payments for training.

Solution

Building Coach Crew was a holistic process involving an amalgamation of UX methods, Interaction design and visual design. We began the project by creating user profiles and personas, based on the information provided to us by the client's interactions with their users. These profiles were then used to create the information architecture for the website, build the framework and define the features and interactions. We kept the designs and the interface in sync with The Coach Crew’s brand voice — simple, bold, and seamlessly direct. View website at www.thecoachcrew.com
